Cursor

Specifies the mouse cursor when pointer is over an element.

Classes
Class Properties
cursor-auto cursor: auto;
cursor-pointer cursor: pointer;
cursor-wait cursor: wait;
cursor-move cursor: move;
Examples
auto
pointer
wait
move
