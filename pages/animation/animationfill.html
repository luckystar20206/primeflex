Animation Fill

Defines how an animation is executed on an element.

Classes
Class Properties
animation-fill-none animation-fill-mode: none;
animation-fill-forwards animation-fill-mode: forwards;
animation-fill-backwards animation-fill-mode: backwards;
animation-fill-both animation-fill-mode: both;
Examples
fill-none
fill-forwards
fill-backwards
fill-both

<div class="card ...">
    <div class="animation-color animation-fill-none ...">fill-none</div>
    <div class="animation-color animation-fill-forwards ...">fill-forwards</div>
    <div class="animation-color animation-fill-backwards ...">fill-backwards</div>
    <div class="animation-color animation-fill-both ...">fill-both</div>
</div>


<style lang="scss">
@keyframes animation-color {
    0%{
        background-color: var(--blue-500)
    }
    100%{
        background-color: var(--green-500)
    }
}
.animation-color {
    animation: animation-color 3s linear;
}
</style>
